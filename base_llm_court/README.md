# LLM Court

LLM Court is a Python project that uses the Gemini API to run a simulated court process with LLM agents acting as the plaintiff's attorney, the defendant's attorney, and the judge.

The case text defined in `main.py` is used as input, and the program generates the court process in the following order:

1. Plaintiff's complaint
2. Defendant's answer
3. Issue identification by the judge
4. Final arguments from both sides
5. Final judgment by the judge

## Project Structure

```text
.
├── main.py      # Entry point. Defines the case text and runs LLMCourt.
├── court.py     # Controls the simulated court flow.
├── agents.py    # Defines the plaintiff, defendant, and judge agents.
├── model.py     # Helper script for listing available Gemini models.
└── README.md
```

## Requirements

- Python 3.10 or later
- Gemini API key
- `google-genai` package

## Setup

Install the required package:

```bash
pip install google-genai
```

Set your Gemini API key as an environment variable:

```bash
export GEMINI_API_KEY="your-api-key"
```

## Usage

```bash
python main.py
```

The program prints each phase of the simulated court process and the final judgment to standard output.

## Changing the Case

To use a different case, edit the `case_text` variable in `main.py`.

```python
case_text = """
Write the case summary, party arguments, evidence, and related facts here.
"""
```

## Models

The current implementation mainly uses the following Gemini models:

- `gemini-2.5-flash`: Generates party arguments and identifies legal issues
- `gemini-2.5-pro`: Generates the final judgment

To list available models, set your API key and run:

```bash
python model.py
```

## Disclaimer

The output generated by this project is simulated legal writing produced by an LLM. It is not a substitute for actual legal advice, legal judgment, or contract review.
